## CN Vocabについて

CN Vocabは、HSK（中国語水平考試）レベル別に効率的に中国語語彙を学習できるアプリです。

### 主な特徴

- HSK 1-6級対応（全2400語）
- スペース反復学習システム
- 習熟度レベル管理
- プライバシー重視設計

### データ安全性

- 個人情報の収集なし
- 学習データは端末内のみに保存
- 第三者への情報共有なし
